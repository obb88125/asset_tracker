# 02. 엑셀 파싱 엔진

## 1. 개요
다양한 형태의 은행 및 증권사 엑셀/CSV 데이터를 통일된 포맷으로 파싱하여 DB에 저장하는 엔진.

## 2. 전체 흐름
1. 파일 업로드 -> `uploads/` 디렉토리에 저장
2. Pandas 리드 (`read_excel` 또는 `read_csv`) -> 상위 10행 미리보기 제공
3. 컬럼 타입 자동 감지 로직 실행 (날짜, 금액 패턴 분석)
4. 사용자가 UI에서 컬럼 매핑 확정 (어느 컬럼이 날짜인지, 금액인지 등)
5. 확정된 매핑 정보를 바탕으로 최종 파싱 및 임포트 수행

## 3. 세부 로직
- **날짜 정규화**: `2024-01-01`, `2024/01/01`, `24.01.01` 등 다양한 문자열을 파싱
- **금액 처리**: 콤마 제거, `천원/만원` 단위 곱셈, (입금/출금) 부호 판별.
- **에러 핸들링**: 파싱에 실패한 행(Row)은 스킵하고, 결과 요약 시 에러 개수를 리포트.

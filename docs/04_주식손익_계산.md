# 04. 주식 손익 계산

## 1. 개요
매수와 매도 내역을 바탕으로 보유 수량, 평균 단가, 실현 손익을 계산하는 로직.

## 2. 계산 알고리즘 (평균단가법)
시간순으로 `StockTrade` 데이터를 순회하며 계산:

- **매수 발생 시:**
  - `총 매수 금액 += 수량 * 단가`
  - `보유 수량 += 매수 수량`
  - `새로운 평균 단가 = 총 매수 누적 금액 / 보유 수량`
- **매도 발생 시:**
  - `보유 수량 -= 매도 수량`
  - `실현 손익 += (매도 단가 - 평균 단가) * 매도 수량 - 수수료 - 세금`

## 3. 포트폴리오 요약
모든 종목의 계산 결과를 합산하여 총 매수 금액, 총 매도 금액, 총 실현 손익을 산출하고 대시보드에 제공.

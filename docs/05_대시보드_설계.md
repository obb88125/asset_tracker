# 05. 대시보드 설계

## 1. 차트 목록
1. **월별 입출금 추이 (Stacked Bar)**: 시간 흐름에 따른 현금 흐름 파악.
2. **인물별 거래 비중 (Doughnut)**: 나와 가장 많이 거래한 상위 N명 비율.
3. **누적 순자산 흐름 (Area Chart)**: 시간이 지남에 따라 순자산(입금-출금)이 어떻게 변해왔는지 누적 그래프.
4. **계좌별 비교 (Grouped Bar)**: 각 은행/증권사별 입출금 총액.
5. **히트맵 (Calendar)**: 일별 거래 빈도 시각화.

## 2. 차트 라이브러리
- **Apache ECharts** 사용.
- 모던하고 깔끔한 다크 모드 렌더링 최적화.
- 화면 크기에 반응형 적용 (`window.addEventListener('resize')`).

## 3. 데이터 집계
- 대부분의 집계 연산은 SQLAlchemy의 `func.sum()`, `func.count()`, `group_by`를 활용하여 DB 단에서 처리 후 반환.
